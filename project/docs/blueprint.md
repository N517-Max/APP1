# **App Name**: CareRoute

## Core Features:

- Transport Request Submission: Users (patients or caregivers) can easily input pickup and dropoff locations, desired date/time, and specify any patient special needs (e.g., wheelchair, oxygen support).
- Driver Matching & Assignment: The system identifies and assigns the most suitable and available driver based on the transport request's requirements and real-time driver status.
- Real-time Trip Tracking: Patients/caregivers can view the assigned vehicle's live location on a map and receive real-time estimated arrival times until the destination is reached.
- In-App Secure Messaging: Facilitates direct and secure communication between the patient/caregiver and the assigned driver, allowing for last-minute clarifications or updates.
- Driver Management & Availability: Drivers can manage their personal profiles, update vehicle details, and control their availability status (online/offline) to accept new requests.
- AI-Powered Route & Assignment Optimizer: An AI tool that suggests optimal routes for drivers and intelligently matches transport requests with the best available driver, considering real-time traffic, patient needs, and driver proximity.

## Style Guidelines:

- Color Palette: Focused on trust and calm with a light scheme.
- Primary color: A serene, professional blue (#2273C3) for conveying reliability and care.
- Background color: A subtle, clean light blue-grey (#F0F2F4), almost white, ensuring readability and a clinical feel.
- Accent color: A deep, authoritative indigo (#2626D9) for key interactive elements and calls to action, providing clear contrast.
- Headline and body font: 'Inter', a grotesque-style sans-serif, chosen for its modern, objective, and neutral appearance, ensuring clear readability across all text.
- Use clear, universally recognizable line-style icons for medical and navigation-related functionalities, promoting quick comprehension.
- A clean, intuitive, and mobile-first layout with clear information hierarchy, emphasizing ease of use for quick transport requests and tracking.
- Subtle, non-distracting animations for map updates, request confirmations, and loading states, enhancing the user experience without visual clutter.
